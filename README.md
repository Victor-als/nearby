- ## Tenologias usadas no projeto.

- React native e Expo
- TypeScript


## Para rodar o projeto localmente

1. Clone o repositório:

```sh
$ git clone https://github.com/Victor-als/nearby
```

3. Instale as dependências:

```sh
npm install
```
4. Rodar localmente:

```sh
npx expo start
```
